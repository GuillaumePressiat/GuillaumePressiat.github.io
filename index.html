PMSI et autres, avec R

Ce site rassemble des informations autour du package R pmeasyr.

Le développement de ce package a été initié au DIM du siège de l'AP-HP. Il était auparavant hébergé sur le github IM-APHP.

  • Le package est hébergé sur Github de GuillaumePressiat
  • Un livret numérique présente le package et son utilisation
  • Ce livret est aussi disponible au format pdf
  • Le manuel pdf avec l'aide pour toutes les fonctions du package
  • Une carte mentale présente les travaux connexes à pmeasyr : Mind map

La partie blog informe des nouveautés du package et rassemble des astuces inhérentes au traitement des données PMSI.
