Un livret numérique documentant l’import et l'étude des données PMSI avec pmeasyr dans R est disponible ici.


Installation du package

devtools::install_github('IM-APHP/pmeasyr')

Cette commande lance l’installation du package et de ses dépendances.


Le projet R du package est hébergé sur le github de l'Information Médicale de l'Assistance Publique - Hôpitaux de Paris : IM-APHP.
