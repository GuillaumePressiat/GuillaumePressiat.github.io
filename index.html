PMSI et autres, avec R

Ce site rassemble des informations autour du package R pmeasyr mais aussi d'autres posts divers sur les données et leur traitement avec R et python

Le développement du package pmeasyr a été initié au DIM du siège de l'AP-HP. Il était auparavant hébergé sur le github IM-APHP.

  • Le package est hébergé sur le github de GuillaumePressiat
  • Un livret numérique présente le package et son utilisation
  • Ce livret est aussi disponible au format pdf
  • Le manuel pdf avec l'aide pour toutes les fonctions du package
  • Une carte mentale présente les travaux connexes à pmeasyr : Mind map

La partie blog informe des nouveautés du package et rassemble des astuces inhérentes au traitement des données PMSI et d'autres posts sur la programmation en général avec R et python
